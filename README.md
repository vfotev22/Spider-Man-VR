## Hardware Requirements:
Ocuus Quest 2 / 3 / 3s

## Target Audience:
This game is designed for players who enjoy movement-heavy VR experiences. 

## Game Synopsis:
In this VR game, you will take on the role of Spider-Man. 
The main gameplay loop includes: 
- Exploring New York City
- Swinging between buildings using web swinging
- Traveling to crime locations to defeat enemies
- Taking pictures for J. Jonah Jameson to earn money for rent.
